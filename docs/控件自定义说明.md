# JSON控件编写规则

## 基本结构

JSON文件是一个控件数组，每个控件对象包含以下基本属性：

```json
[
    {
        "Type": "控件类型",
        "Name": "控件名称",
        "Parent": "父控件名称(可选)",
        "Label": "控件标签文本",
        "Style": ["样式列表"],
        "Event": {"事件名": {"操作类型": "参数"}},
        "Size": [ "宽度", "高度"],
        "Position": ["X坐标", "Y坐标"],
        "Sizer": "布局管理器(可选)"
    }
]
```

## 属性说明

1. **Type**（必需）
   - 控件类型，如：Button, TextCtrl, StaticText等

2. **Name**（必需）
   - 控件的唯一标识名称
   - 示例：`"btnOK"`

3. **Parent**（可选）
   - 父控件的名称
   - 示例：`"panel"`

4. **Label**（可选）
   - 控件显示的文本
   - 示例：`"确定"`

5. **Style**（可选）
   - 控件样式列表，可以是以下值：
     - `"wx.ALIGN_CENTER"`: 居中对齐
     - `"wx.ALIGN_LEFT"`: 左对齐
     - `"wx.ALIGN_RIGHT"`: 右对齐
     - `"wx.EXPAND"`: 扩展填充
   - 示例：`["wx.ALIGN_CENTER", "wx.EXPAND"]`

6. **Event**（可选）
   - 事件绑定，格式为：
   ```json
   "Event": {
       "事件名": {
           "操作类型": "参数"
       }
   }
   ```
   支持的事件类型：
   - `"ButtonClick"`: 按钮点击
   - `"TextEnter"`: 文本框回车
   - `"MouseDClick"`: 鼠标双击
   
   支持的操作类型：
   - `"OpenLink"`: 打开链接
     - 参数：URL地址
     - 示例：`{"ButtonClick": {"OpenLink": "https://www.example.com"}}`
   - `"OpenFile"`: 打开文件
     - 参数：文件路径
     - 示例：`{"ButtonClick": {"OpenFile": "C:\\test.txt"}}`
   - `"OpenFolder"`: 打开文件夹
     - 参数：文件夹路径
     - 示例：`{"ButtonClick": {"OpenFolder": "C:\\Documents"}}`
   - `"RunCommand"`: 执行命令
     - 参数：命令字符串
     - 示例：`{"ButtonClick": {"RunCommand": "dir"}}`
   - `"OpenApp"`: 打开应用程序
     - 参数：应用程序路径
     - 示例：`{"ButtonClick": {"OpenApp": "notepad.exe"}}`
   - `"RunPythonCode"`: 执行Python代码
     - 参数：Python代码字符串
     - 示例：`{"ButtonClick": {"RunPythonCode": "print('Hello World!')"}}`

7. **Size**（可选）
   - 控件尺寸，格式：[宽度, 高度]
   - 示例：`[100, 30]`

8. **Position**（可选）
   - 控件位置，格式：[X坐标, Y坐标]
   - 示例：`[10, 20]`

9. **Sizer**（可选）
   - 布局管理器类型
   - 示例：`"wx.BoxSizer"`

## 示例

```json
[
    {
        "Type": "Button",
        "Name": "btnOK",
        "Label": "确定",
        "Size": [100, 30],
        "Position": [10, 10],
        "Event": {
            "ButtonClick": {
                "OpenLink": "https://www.example.com"
            }
        }
    },
    {
        "Type": "TextCtrl",
        "Name": "txtInput",
        "Size": [200, 25],
        "Position": [10, 50],
        "Event": {
            "TextEnter": {
                "RunCommand": "echo Hello"
            }
        }
    }
]
```

## 注意事项

1. 所有字符串值都需要使用双引号
2. 数值和布尔值不需要引号
3. 事件处理函数中的参数值会自动转换为相应的类型
4. 如果未指定某些可选属性，将使用默认值
5. 控件的创建顺序与JSON数组中的顺序一致

## 错误处理

如果JSON格式不正确或缺少必需属性，程序会抛出TypeError异常。建议在编写JSON文件时使用JSON验证工具进行检查。
