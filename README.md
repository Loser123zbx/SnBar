# SnBar   锡条

## 做着玩的小项目，一个菜单栏。
### English [README.EN]()

### 一个简单的任务栏工具栏
- 支持自定义布局
- 支持自定义控件
- 支持自定义图标
- 支持自定义界面

## 使用方法
### 直接下载Release -- 基础自定义 -- 适合小白

自定义控件教程详见[自定义控件教程]()

可自定义控件、布局、图标、界面，以及预设的整合控件,在`RunPythonCode`运行函数中使用python内置库及以下库:
- wxpython
- keyborad
- requests

### 下载源代码后打包  -- 高级自定义 -- 适合有一定python基础的用户
